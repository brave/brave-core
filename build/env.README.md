# Brave environment setup helpers

The directory contains shell helpers to set Brave-specific environment
variables. These variables allows you to run `gn`, `autoninja`, `gclient` and
other tools directly.

These helpers are NOT required to run builds using `run build/test` commands.

There're 3 versions to cover most popular interpreters:

- `env.cmd` (cmd.exe)
- `env.ps1` (powershell)
- `env.sh` (cygwin/bash/zsh)

Suported parameters:

- `-v` enable verbose output.

If you modify one of the scripts, make sure to do the same changes in other
versions.

Variables are generated by running `node build/commands/scripts/genEnv.js`
command.
